# Mami Signature CRM

## Project Overview
Mami Signature CRM is a comprehensive customer relationship management system designed to manage and streamline interactions with clients. It offers a variety of tools for tracking leads, managing customer data, and automating sales processes. The goal of Mami Signature CRM is to enhance customer engagement and improve business outcomes through effective management of client relationships.

## Setup Instructions
1. **Clone the repository:**  
   Run the following command in your terminal:
   ```bash
   git clone https://github.com/Ofentsedee/mami-signature-crm.git
   ```  

2. **Install dependencies:**  
   Navigate to the project directory and install the required packages:
   ```bash
   cd mami-signature-crm
   npm install  
   ```  

3. **Configuration:**  
   Create a `.env` file in the root directory and specify the necessary environment variables such as database connection strings and API keys.

4. **Start the application:**  
   Use the following command to run the application:
   ```bash
   npm start  
   ```
